# 🎤 Zingen

> Plataforma de karaokê moderna, responsiva e estilizada — feita com HTML e CSS puro.

---

## 📌 Sobre o Projeto

O **Zingen** é uma landing page completa de um app de karaokê, desenvolvida com foco em boas práticas de CSS moderno. O projeto explora layouts avançados, responsividade real e componentes visuais sofisticados — tudo sem JavaScript.

---

## ✨ Técnicas e Recursos Utilizados

- **CSS Grid** — layout de múltiplas colunas e linhas com `grid-template-columns` e `grid-template-rows`
- **Grid Template Areas** — posicionamento preciso dos cards com `grid-column` e `grid-row`
- **Responsividade com `@media`** — adaptação do layout para mobile e desktop
- **CSS Custom Properties** — variáveis para cores, tipografia e espaçamentos
- **CSS Nesting** — estilização aninhada de componentes de forma organizada
- **Flexbox** — alinhamento e distribuição de elementos
- **`padding-block` e `padding-inline`** — espaçamentos modernos com lógica de escrita
- **`min()` no container** — largura fluida com limite máximo responsivo
- **Gradientes lineares** — bordas e fundos com `linear-gradient`
- **`position: absolute`** — posicionamento de badges e imagens decorativas
- **Web Component customizado** — elemento `<zin-pricing>` estilizado via CSS
- **Ícones SVG via CSS** — troca de ícones sociais no hover com variáveis CSS
- **`radial-gradient`** — separadores visuais entre seções
- **`inset`** — atalho moderno para posicionamento absoluto

## 📄 Seções da Página

| Seção | Descrição |
|---|---|
| **Hero** | Apresentação principal com ilustração |
| **#features** | Grid de cards com funcionalidades do app |
| **#pricing** | Planos com card premium em destaque |
| **#download** | CTA para download do app |
| **footer** | Navegação, redes sociais e créditos |
